HTML & CSS Basic Summary

HTML Summary

HTML (HyperText Markup Language) is used to define a page content, structure and meaning. HTML is not used for styling purposes, CSS (Cascading Style Sheet) is used instead.

  • HTML uses "elements" to describe (annotate) contents.
  • HTML elements typically have an opening tag, content and a closing tag
  • There is the void (empty) elements e.g images.
  • Elements can be configured with attributes.
  • There is a long list of available elements, but experience will be gained over time, no worries!

Learn more about all available HTML elments on the MDN HTML element reference. .

CSS Summary

CSS ( Cascading Style Sheets) is used for styling a page content.

  • Styles are assigned via property-value pairs.
  • Styles can be assigned via the "style" attributes.
  • To avoid code duplication, global styles one typically used (e.g via the "style" element) instead.
  • Alternatively, external style sheet files can be worked with which can "link" to.
  • When working with CSS, concepts such as "Inheritance", "Specificity", and "Box Model" should be understood.

Learn more about all available CSS properties and values on the MDN CSS property reference.
